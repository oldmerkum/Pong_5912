# Pong_5912

## Game_Description:

  A basic pong implementation.
  * 1P vs 2P
  * 1P vs AI
  * Debug Mode
  * Screen Capturing
  * Video Recording

## Controls:

  Player One Movement: 
  
    Up/Down -> W/S
    
  Player Two Movement: 
  
    Up/Down -> ArrowUp/ArrowDown
    
  Saving:
  
    Screen Capture -> C
    
    Video Recording -> R
    
  Debug Mode:
  
    Show/Hide Debug Menu -> T
    
    Change AI Paddle Position -> 1,2,3
